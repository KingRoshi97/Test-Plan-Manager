# POL-0a — Boundary Checklist

Belongs in `policy/` if it answers:
- "What does PROD/COMPLIANCE require?"
- "When can we override a gate or quota?"
- "Who can approve and what must be recorded?"
- "How do conflicting rules resolve?"

Does NOT belong in `policy/` if it answers:
- "How do we call policy?" (system)
- "How do we evaluate predicates?" (gates)
- "What stages exist?" (orchestration)
