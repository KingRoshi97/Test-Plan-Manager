# STD-0a — Boundary Checklist

Belongs in `standards/` if it answers:
- "What standards packs exist and what rules do they contain?"
- "Which standards apply for this run context?"
- "How do we resolve conflicting standards deterministically?"
- "What does the resolved snapshot look like?"

Does NOT belong in `standards/` if it answers:
- "How do we evaluate gates?" (gates)
- "What templates do we render?" (templates)
- "What is the canonical spec schema?" (canonical)
- "How is policy invoked?" (system)
