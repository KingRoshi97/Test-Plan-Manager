# CAN-5b — Determinism Rules (Artifacts)
- Canonical spec content is deterministic (CAN-1..CAN-3).
- Unknowns artifact content is deterministic (CAN-4).
- Build report counts are derived from those pinned artifacts only.
- Notes in build report must be stable (avoid timestamps, random phrasing).
