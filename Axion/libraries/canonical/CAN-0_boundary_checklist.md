# CAN-0a — Boundary Checklist

Belongs in `canonical/` if it answers:
- "What is the canonical representation of the target system?"
- "How are entities identified and related?"
- "What is an unknown/assumption and how is it recorded?"
- "What integrity rules must the canonical spec satisfy?"

Does NOT belong in `canonical/` if it answers:
- "What are the intake questions?" (intake)
- "Which standards apply?" (standards)
- "How do we render templates?" (templates)
- "How do we evaluate gates?" (gates)
