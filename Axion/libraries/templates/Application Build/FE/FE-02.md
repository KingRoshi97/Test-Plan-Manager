## 2. Purpose

Create the canonical inventory of all UI components (shared and screen-specific), including component_id, props interface, events emitted, state bindings, and screen usage. This template drives composition, theming, accessibility, and state management.

## 10. Completeness Gate

- Gate ID: TMP-05.PRIMARY.FE
- Pass conditions:
  - [ ] required_fields_present == true
  - [ ] all component_ids are unique
  - [ ] screen_usage binds to valid screen_ids
  - [ ] placeholder_resolution == true
  - [ ] no_unapproved_unknowns == true
