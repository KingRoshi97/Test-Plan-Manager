# TMP-0a — Boundary Checklist

Belongs in `templates/` if it answers:
- "What templates exist and where are they?"
- "How do we select templates deterministically?"
- "What placeholders/sections are required?"
- "How do we validate template completeness?"

Does NOT belong in `templates/` if it answers:
- "What is the canonical spec schema?" (canonical)
- "What standards apply?" (standards)
- "How do gates evaluate predicates?" (gates)
- "How do we package the kit?" (kit)
