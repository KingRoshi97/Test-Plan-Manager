# Intellectual Property Policy — Templates Library

## Purpose

Defines IP rules for template content, including authorship, reuse, and attribution requirements.

## Core Rules

### Authorship

- All templates are authored internally and owned by the templates library owner
- Templates must NOT contain third-party copyrighted content without license verification
- Example and placeholder content must use clearly synthetic values

### Reuse

- Templates may be freely rendered and filled during builds
- Template structure and placeholder patterns are internal assets
- Rendered outputs are original work shaped by templates, not copies of templates

### Secrets and Sensitive Data

- Templates must NOT contain actual secrets, API keys, passwords, or credentials
- Templates must NOT contain real PII
- Placeholder values referencing sensitive data must use synthetic examples
