# Testing & QA

Covers test pyramid, deterministic testing, test data strategy, contract testing, golden fixtures, and property-based testing.

## Subfolders

- **concepts/** — foundational what/why knowledge
- **patterns/** — reusable approach guidance (pattern-only)
- **procedures/** — step-by-step how-to instructions
- **checklists/** — enforceable verification lists
- **pitfalls/** — common failure modes and anti-patterns
- **references/** — tables, defaults, parameter lists
- **examples/** — structure learning examples (restricted)
