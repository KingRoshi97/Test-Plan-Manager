# IT End-to-End Bundles

  ## Overview
  Bundles group related KIDs for common use cases within the IT_END_TO_END pillar.

  ## Available Bundles
  - **Security Baseline**: Core security KIDs applicable to all builds
  - **API Service Kit**: Networking + API + auth KIDs for API services
  - **Full-Stack Web Kit**: Frontend + backend + infra KIDs for web applications
  - **Data Layer Kit**: Database + caching + storage KIDs for data-heavy applications
  - **Platform Ops Kit**: Cloud + IAM + compliance + release KIDs for platform teams

  ## Bundle Selection
  Bundles are selected based on run profile and risk class. Multiple bundles may be combined for a single run.
