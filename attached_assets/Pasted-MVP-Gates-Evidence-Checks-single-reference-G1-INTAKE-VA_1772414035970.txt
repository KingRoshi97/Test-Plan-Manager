G8_PACKAGE_INTEGRITY (Stage: S10_PACKAGE)

Evidence files

.axion/runs/<run_id>/kit/packaging_manifest.json

.axion/runs/<run_id>/kit/bundle/** (the files referenced in the manifest)

Checks (v1)

packaging manifest exists

valid JSON

algorithm == "sha256"

for each files[] entry:

safe relative path

file exists in bundle root

sha256(file bytes) matches manifest

Implementation hint (keeps everything uniform)

Implement gates as ordered check lists with these primitive ops:

file_exists(path)

json_valid(path)

json_has(path, pointer)

equals(path, pointer, value) (for algorithm / hard_failures)

gte(path, pointer, min) (coverage)

verify_hash_manifest(manifest_path, bundle_root)

This is enough to make the system block correctly without a full DSL.
