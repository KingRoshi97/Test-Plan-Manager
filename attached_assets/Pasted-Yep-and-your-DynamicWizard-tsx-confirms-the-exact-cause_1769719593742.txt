One more important note (ZIP gating)

Your ZIP gating currently only checks isPackageReady when the step id is "upload": 

DynamicWizard

if (currentStep?.id === "upload") { ... }


If your zip step id is not literally "upload" (some flows use "zip_required" etc), that gating won’t trigger.

Fix it like this:

const isZipStep = currentStep.fieldGroups
  .flatMap(g => g.fields)
  .some(f => f.ui === "file_zip");

if (isZipStep) {
  const zipRequired = currentStep.fieldGroups
    .flatMap(g => g.fields)
    .some(f => f.ui === "file_zip" && f.required);

  if (zipRequired && !isPackageReady) return false;
}


This makes ZIP gating work regardless of step id.
