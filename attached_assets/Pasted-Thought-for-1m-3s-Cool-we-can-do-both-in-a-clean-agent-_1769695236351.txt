2) Brand token lock-in (based on your logo)

From your logo images, the dominant brand colors cluster around:

Amber/Gold: #E0A040

Orange: #E08040

Warm Orange: #E06040

Red-Orange: #E04020

Red: #E02020

That’s a premium “warm-metal + flame gradient” identity. The key is use the gradient sparingly.

Token strategy

Primary accent (solid): Amber #E0A040

Brand gradient: Amber → Orange → Red-Orange

Dark neutrals: deep charcoal layers (not pure black)

Light theme: clean whites with warm accents

Drop-in CSS variables (shadcn-style, HSL)

Put this in your global CSS (where you set :root and .dark).
You can paste this into client/src/index.css under your current variables.

How to use the gradient (rules)

Use gradient only for:

Primary CTA button background

Tiny accent lines/dividers

Small badges (optional)
Avoid gradient for:

Full page backgrounds

Long text sections

Tables

You can implement as a utility:

.bg-axiom-gradient {
  background-image: linear-gradient(
    90deg,
    hsl(var(--axiom-amber)) 0%,
    hsl(var(--axiom-orange)) 45%,
    hsl(var(--axiom-redorange)) 100%
  );
}
