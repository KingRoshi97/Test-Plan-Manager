# Axion App

This application was scaffolded by AXION from locked documentation.

## Development

```bash
npm install
npm run dev
```

## Stack

- Frontend: React + TypeScript
- Backend: Express.js + TypeScript
- Database: PostgreSQL + Drizzle ORM
- Deployment: Replit
